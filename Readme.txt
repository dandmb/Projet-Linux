Bonjour,
L'utilisation de ce script est très simple;

Téléchargez le dossier de base du projet avec la commande :

git clone https://github.com/dandmb/Projet-Linux-admin.git

ensuite executez la commande

./script.sh build dossier/dossier.....

pour telecharger et creer les fichiers complementaires

et enfin placez vous dans dossier/dossier.....

et executez la commande ./script.sh -h (ou --help) pour afficher les differentes fonctionnalités du script.
